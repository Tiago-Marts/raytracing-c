# Introdução
Aplicação dos conceitos vistos em ["Ray Tracing in One Weekend"](https://raytracing.github.io/books/RayTracingInOneWeekend.html)  para a linguagem C

# Executando
Para gerar o executável, execute o makefile. Então, para gerar a imagem resultante, execute o executavel "raytracing.exe" gerado na pasta output/bin
